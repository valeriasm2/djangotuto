# Django Tutorial Project

Este proyecto es una aplicación de ejemplo en Django llamada `djangotutorial`. Aquí se explica cómo ejecutar el proyecto en un entorno local.

---

## Requisitos

- Python 3
- Virtualenv (opcional pero recomendado)
- Django instalado en el virtualenv

---

## Instalación y ejecución

1. **Activar el entorno virtual** (asegúrate de estar en la carpeta correcta):

```bash
source ../envpolls/bin/activate
```

2. **Ejecutar el servidor de desarrollo de Django:**

```bash
python3 manage.py runserver
```

3. **Abrir la aplicación en el navegador:**
Visita la url para ver la aplicación funcionando.
```bash
 http://127.0.0.1:8000/admin
```
